Select one of the projects above and complete the project steps below. All code must be executed in an Oracle database.

1. Draw an E-R diagram for your project (it must contain at least 12 tables). Use Microsoft Visio or a similar software tool.
2. Provide the table creation scripts for your project.
3. Develop the scripts that create your project's indexes (primary, foreign, or composite keys).
4. Implement data insertion for your project using a procedure or function that can be called from a PL/SQL package.
5. Implement data updates for your project using a procedure or function that can be called from a PL/SQL package.
6. Implement data deletion for your project using a procedure or function that can be called from a PL/SQL package.
7. Using a PL/SQL trigger, ensure that when data is inserted into a reference table, corresponding data is also inserted into a transaction table.
8. To support reporting, write multiple query code snippets using PL/SQL. Pass dynamic filters to the WHERE clause of SELECT statements.
9. Demonstrate that the database in your project's E-R diagram is in Third Normal Form (3NF).
10. Use a procedure or function callable from a PL/SQL package to delete duplicate records from tables.
